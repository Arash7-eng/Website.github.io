üìò About HTML

HTML stands for HyperText Markup Language. It is the standard language used to create and design webpages.

HTML uses a series of tags to structure content such as text, images, links, and forms. These tags tell the browser how to display the content.

Key points about HTML:

  • It is not a programming language. It is a markup language.
  • HTML files end with the .html or .htm extension.
  • It is supported by all modern web browsers.
  • HTML can be combined with CSS for styling and JavaScript for behavior.

üî§ HTML Elements

Here are some common HTML elements:

  • <h1> to <h6> - Headings (h1 is largest, h6 is smallest)
  • <p> - Paragraph
  • <a> - Anchor/Link
  • <img> - Image
  • <ul>, <li> - Unordered List and List Items
  • <ol>, <li> - Ordered List and List Items
  • <form> - Form
  • <input> - Input Field
  • <textarea> - Text Area
  • <table> - Table
  • <div> and <span> - Layout containers
