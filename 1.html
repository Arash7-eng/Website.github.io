Introduction of HTML

1. What is HTML?

HTML means HyperText Markup Language. It is used to create web pages. It shows text, images, links, and other content in a browser.

2. Full Form and Meaning

  • H - HyperText: Text that links to other pages
  • M - Markup: Tags used to format the content
  • L - Language: A way to communicate with the browser

3. Structure of an HTML Page

  1. <!DOCTYPE html> - Tells the browser it is an HTML5 file
  2. <html> - Root tag of the page
  3. <head> - Page information like title
  4. <body> - Main content of the page

4. Common HTML Tags

  • <h1> to <h6>: Headings
  • <p>: Paragraph
  • <ul>, <ol>, <li>: Lists
  • <a>: Links
  • <img>: Images

5. HTML with CSS, JavaScript, and DOM

  • HTML: Builds the page structure
  • CSS: Adds styles like color, font, and layout
  • JavaScript: Adds interaction like button clicks
  • DOM: Allows JavaScript to update HTML content live
